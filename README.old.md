# Furniture E-commerce Platform

This is a custom, scalable e-commerce application for selling furniture online. 
It features a modern, headless architecture with a clean frontend, robust backend, 
and fully functional e-commerce features.

Key features:
- Product catalogue with categories, filters, and search
- User authentication and roles (customers, admin)
- Shopping cart and checkout with Stripe payments
- Admin dashboard for managing products, orders, and users
- Modern responsive design for desktop and mobile
- Optimised for SEO and performance

Tech stack:
- Frontend: Next.js, React, TypeScript, Tailwind CSS, Framer Motion
- Backend: NestJS, Node.js, PostgreSQL, Prisma
- Payments: Stripe
